## Usage

The Link component is used for navigation and actions that navigate to other pages or external resources.

## Props

The Link component accepts the following props:

- `size`: Defines the size of the link (`"small"` | `"medium"` | `"large"`, default: "medium")
- `external`: Boolean to indicate if the link opens in a new tab
- `href`: The URL that the link points to
- `className`: Additional CSS class names
- All standard HTML anchor (`<a>`) attributes

## External Links

Links can be configured to open in a new tab with proper security attributes.

<Playground
  scope={{ Link }}
  code={`
<Link href="https://example.com" external>
  External Website
</Link>
  `}
/>

## Accessibility

The Link component follows accessibility best practices:

- Uses semantic HTML `<a>` element
- Maintains proper color contrast
- Includes proper focus states
- Adds appropriate attributes for external links
- Supports keyboard navigation
